**SageMovies** is a Netflix-inspired streaming platform that provides free access to movies. Built with modern web technologies, it delivers a seamless streaming experience while maintaining a familiar and intuitive user interface.

Developed with **Node.js** and styled with **Tailwind CSS**, deployed on Vercel for optimal performance and scaling.

## Key Features

- **Free Movie Streaming**
  - Extensive library of free movies
  - High-quality video playback
  - No subscription required
  - User-friendly interface

- **Netflix-Style Experience**
  - Responsive grid layout
  - Movie categorization
  - Dynamic content loading
  - Smooth animations and transitions

- **User Features**
  - Movie search functionality
  - Genre-based filtering
  - Watchlist management
  - Viewing history
